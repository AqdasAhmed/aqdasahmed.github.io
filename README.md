# My Personal Portfolio

Welcome to my personal portfolio website! This is a React-based portfolio showcasing my projects, skills, and experiences.

## 🚀 Live Demo

Visit the live site: [aqdasahmed.github.io](https://aqdasahmed.github.io)

## 🛠️ Tech Stack

- **Frontend:** React, HTML, CSS, JavaScript
- **Styling:** CSS Grid & Flexbox
- **Deployment:** GitHub Pages

## 📦 Installation

```bash
# Clone the repository
git clone https://github.com/AqdasAhmed/aqdasahmed.github.io.git

# Navigate to the project directory
cd aqdasahmed.github.io

# Install dependencies
npm install

# Start development server
npm start
```

## 🏗️ Building for Production

```bash
# Create production build
npm run build
```

The build output will be in the `build/` folder, ready for deployment to GitHub Pages.
